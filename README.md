# Pugbot
Pickups bot 
# Engines
 - `npm - v6.14.8^`
 - `node - v14^`
 - Uses ECMAScript 2020
# Installation
- Make a file called `.env` and add `TOKEN=YOUR_DISCORD_BOT_TOKEN` to it.
- Start a MongoDB server or spawn a cluster in Atlas and add `DB=YOUR_CONNECTION_STRING` to `.env`
- run `npm ci`
- run `npm start`
